# Life Dashboard

Personal productivity cockpit built with React + Vite + Tailwind CSS + Recharts.

## Setup

```bash
npm install
npm run dev
```

## Pages

- **Dashboard** — Morning overview with widgets from all domains
- **Workout Tracker** — Exercise database, logging, progression charts
- **Body Tracker** — Weight, measurements, body fat % trends
- **Study Checklist** — Daily learning tracks with micro-logs and streaks
- **Budget Tracker** — Income/expenses, category charts, budget targets
- **Job Applications** — Table/kanban views, status pipeline, stats
- **Habit Streaks** — 90-day heatmaps, weekly scorecard, trend charts
- **Quick Capture** — Zero-friction inbox for thoughts and todos

## Stack

React 18 · Vite · Tailwind CSS 3 · Zustand · Recharts · Framer Motion · Lucide Icons
